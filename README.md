# 📊 Ecom Express Sales Dashboard

An interactive Power BI dashboard developed to analyze and visualize Ecom Express sales performance, revenue trends, cancellation analysis, category insights, and state-wise business growth.

---

## 🚀 Project Overview

The **Ecom Express Sales Dashboard** helps businesses monitor sales activities and gain meaningful insights from e-commerce data.  
This dashboard provides a complete overview of:

- Revenue Performance
- Average Order Value (AOV)
- Total Sales
- Cancellation Analysis
- Product-wise Revenue
- State-wise Revenue Distribution
- Monthly Revenue Trends

The dashboard is fully interactive with category filters and dynamic visualizations.

## 📈 Key Insights from Dashboard

- Total Revenue reached **₹1.25bn**
- Total Sales recorded **₹1.77bn**
- Average Order Value (AOV) is **₹112.85K**
- Maharashtra generated the highest revenue among all states
- Laptop category contributed the maximum revenue
- Cancellation rate observed was **29.72%**
- Peak revenue was generated around mid-2024
